## 🔑 关键概念

### 闲时计算
- 在系统资源空闲时（CPU < 20%, 内存 < 70%, 充电）
- 批量处理非紧急任务（Embedding, VLM, ASR）
- 串行加载模型，避免内存爆炸
- 1 小时最多执行 1 次

### 模型分类
- **实时处理**: OCR (50MB 常驻)
- **按需加载**: LLM (2.5GB, 2 分钟后卸载)
- **闲时处理**: Embedding (650MB), VLM (3GB), ASR (80MB)

### 内存优化
- 原方案: 13.5GB 常驻
- 新方案: 50MB - 3GB
- 节省: 85%+

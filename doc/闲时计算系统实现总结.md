## 使用方法

### 1. 安装依赖

```bash
pip install psutil
```

### 2. 运行示例

```bash
cd ai-sidecar
python idle_compute_example.py
```

### 3. 集成到 AI Sidecar

```python
from idle_compute import (
    IdleDetector,
    TaskScheduler,
    ModelManager,
    IdleComputeEngine,
    Task,
    TaskType,
    TaskPriority,
    ModelType,
)

# 创建组件
idle_detector = IdleDetector(
    cpu_threshold=20.0,
    memory_threshold=70.0,
    stable_duration=60,
    min_interval=3600,
)

task_scheduler = TaskScheduler()
model_manager = ModelManager(max_concurrent_models=2)

# 注册模型
model_manager.register(
    ModelType.OCR,
    name="PaddleOCR-INT8",
    loader=load_ocr_model,
    memory_mb=50,
    keep_loaded=True,  # 常驻
)

model_manager.register(
    ModelType.EMBEDDING,
    name="BGE-M3-INT8",
    loader=load_embedding_model,
    memory_mb=650,
    keep_loaded=False,  # 闲时加载
)

# 创建引擎
engine = IdleComputeEngine(
    idle_detector=idle_detector,
    task_scheduler=task_scheduler,
    model_manager=model_manager,
)

# 启动
await engine.start()

# 提交任务
task = Task(
    task_id="embed-1",
    task_type=TaskType.EMBEDDING,
    priority=TaskPriority.IDLE_HIGH,
    payload={"capture_id": 1, "text": "测试文本"}
)
task_scheduler.submit_task(task)
```

## 核心特性

### 1. 智能闲时检测
```python
检测条件（全部满足）:
- CPU < 20% (持续 1 分钟)
- 内存 < 70% (持续 1 分钟)
- 笔记本在充电
- 距离上次执行 > 1 小时
```

### 2. 串行模型加载
```python
闲时处理流程:
1. 加载 BGE-M3 (650MB)
2. 批量处理 100 条 Embedding
3. 卸载 BGE-M3
4. 加载 MiniCPM-V (3GB)
5. 批量处理 10 条 VLM
6. 卸载 MiniCPM-V
7. 加载 Whisper (80MB)
8. 批量处理 20 条 ASR
9. 卸载 Whisper
```

### 3. 内存占用控制
```python
正常时间:
  PaddleOCR: 50MB (常驻)
  总计: 50MB

用户提问时:
  PaddleOCR: 50MB
  Qwen2.5: 2.5GB (按需加载)
  总计: 2.5GB

闲时计算:
  PaddleOCR: 50MB
  BGE-M3: 650MB (或)
  MiniCPM-V: 3GB (或)
  Whisper: 80MB
  峰值: 3GB (串行加载)
```

## 配置参数

### 闲时检测器

| 参数 | 默认值 | 说明 |

### 3. 夜间批量处理

```python
from datetime import datetime

async def _idle_worker(self):
    while self._running:
        await asyncio.sleep(10)

        # 夜间模式（23:00 - 06:00）
        hour = datetime.now().hour
        if 23 <= hour or hour < 6:
            # 放宽条件
            if self._check_resource_state_relaxed():
                await self._process_idle_tasks()
        else:
            # 正常模式
            if self.idle_detector.is_idle():
                await self._process_idle_tasks()
```

## 故障排查

### 问题 1: 闲时任务从不执行

**原因**: 系统资源从未满足条件

**解决**:
```python
# 检查状态
status = idle_detector.get_status()
print(status)

# 放宽阈值
idle_detector.cpu_threshold = 30.0
idle_detector.memory_threshold = 80.0
```

### 问题 2: 内存占用过高

**原因**: 模型未正确卸载

**解决**:
```python
# 强制卸载所有模型
model_manager.unload_all(force=True)

# 检查状态
status = model_manager.get_status()
print(f"已加载: {status['loaded_count']}")
```

### 问题 3: 任务堆积

**原因**: 闲时执行频率太低

**解决**:
```python
# 缩短执行间隔
idle_detector.min_interval = 1800  # 30 分钟

# 或清空队列
task_scheduler.clear_idle_queue()
```

```python
# ai-sidecar/dispatcher.py

from idle_compute import TaskScheduler, Task, TaskType, TaskPriority

class Dispatcher:
    def __init__(self):
        self.task_scheduler = TaskScheduler()
        # ...

    async def dispatch(self, req: IpcRequest) -> IpcResponse:
        if req.task.type == "embed":
            # 提交到闲时队列
            task = Task(
                task_id=req.id,
                task_type=TaskType.EMBEDDING,
                priority=TaskPriority.IDLE_HIGH,
                payload=req.task.dict()
            )
            self.task_scheduler.submit_task(task)
            return IpcResponse.make_ok(req.id, {"status": "queued"})
```

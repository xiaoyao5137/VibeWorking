#### 数据库结构

```sql
CREATE TABLE knowledge_entries (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    capture_id INTEGER NOT NULL,
    summary TEXT NOT NULL,
    entities TEXT,                    -- JSON 数组
    category TEXT,                    -- 会议/文档/代码/聊天/其他
    importance INTEGER DEFAULT 3,     -- 1-5
    user_verified BOOLEAN DEFAULT 0,
    user_edited BOOLEAN DEFAULT 0,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (capture_id) REFERENCES captures(id)
);

-- FTS5 全文搜索索引
CREATE VIRTUAL TABLE knowledge_fts USING fts5(
    summary,
    entities,
    content='knowledge_entries',
    content_rowid='id'
);
```

## API 使用示例

### 获取知识列表

```bash
curl "http://localhost:7070/api/knowledge?limit=10&category=会议"
```

### 搜索知识

```bash
curl "http://localhost:7070/api/knowledge/search?q=产品&limit=20"
```

### 验证知识

```bash
curl -X POST "http://localhost:7070/api/knowledge/1/verify"
```

### 删除知识

```bash
curl -X DELETE "http://localhost:7070/api/knowledge/1"
```

### 提炼知识

```bash
curl -X POST "http://localhost:7070/api/knowledge/extract" \
  -H "Content-Type: application/json" \
  -d '{"capture_id": 123}'
```
